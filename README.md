# Hotkeys for master duel

(Script for master duel)<br>
Provides hotkeys to easily reach often used UI elements.

## Hotkeys :
<pre>
General :
space    : Show card infos
q        : Previous arrow
d        : Next arrow

Deck :
arrow keys : Navigate card list
!        : Add card to favorites

Battle :
s        : Phase switcher
a        : Battle phase
z        : Main phase 2
e        : End phase

Script :
shift+s  : Pause script
shift+q  : Exit script
</pre>
