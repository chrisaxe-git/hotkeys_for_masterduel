# Hotkeys for master duel

(Script for master duel)<br>
Provides hotkeys to easily reach often used UI elements.

Hotkeys :  
- s : Phase switcher  
- a : Battle phase  
- z : Main phase 2  
- e : End phase  
- q : Previous arrow  
- d : Next arrow  
- space : Show card infos  
- shift+s : Pause  
- shift+q : Exit
